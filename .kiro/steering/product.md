# Product Overview

## Project
Beltline Panic is a fast-paced factory survival game built for Gamedev.js Jam 2026.

## Core fantasy
The player is a factory worker holding a fragile machine-driven production loop together under increasing pressure.

## Core loop
- Items move on a looping conveyor belt
- The player moves from the center to machines or the upgrade terminal
- Machine 1 is mandatory for every item
- Machine 2 and Machine 3 are optional and increase item value
- Completed items generate money after surviving the full belt loop
- Money is spent on upgrades and automation
- Throughput increases over time
- Backlog ends the run

## Design priorities
- Readability over realism
- Fast iteration over ambitious scope
- Strong arcade pressure
- Clear machine theme
- Simple systems with strong feedback
- Jam-safe feature choices

## Non-goals
- No multiplayer
- No backend
- No true isometric simulation
- No physics-heavy conveyor system
- No large content scope

## When helping on this project
Prefer solutions that:
- preserve the core loop
- reduce implementation risk
- improve clarity and responsiveness
- fit a solo jam schedule
